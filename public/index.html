Alfresco User API

โปรเจคนี้แยกจากของเดิม ใช้ Alfresco username/password ของ user เพื่อให้ permission การเห็น folder/file เป็นไปตามสิทธิ์ใน Alfresco จริง

ของเดิม alfresco-api ยังใช้ได้เหมือนเดิมที่ port 3000 ส่วนโปรเจคใหม่นี้ default ใช้ port 3001

1. Login

POST http://localhost:3001/auth/login
Content-Type: application/json

{
  "username": "alfresco_user",
  "password": "alfresco_password"
}

Response จะได้ accessToken เอาไปใช้เป็น Bearer token ของ user session

2. List folders ตามสิทธิ์ user

GET http://localhost:3001/user-api/alfresco/folders?path=/Sites/tg-saving/documentLibrary
Authorization: Bearer <accessToken>

3. List/Search documents ตามสิทธิ์ user

GET http://localhost:3001/user-api/alfresco/documents?folderPath=/Sites/tg-saving/documentLibrary/การเงิน&q=026277&maxItems=20&skipCount=0
Authorization: Bearer <accessToken>

4. Open file

GET http://localhost:3001/user-api/alfresco/documents/DOCUMENT_ID/content?name=file.pdf
Authorization: Bearer <accessToken>
